# VisionTrainer Mobile App

## 将来の実装予定

このディレクトリは将来のExpo（React Native）モバイルアプリ用に予約されています。

### 計画されている機能

- iOS/Androidネイティブアプリ
- オフラインサポート
- プッシュ通知リマインダー
- ARを使用した視力トレーニング（将来）

### 技術スタック予定

- Expo SDK 52+
- React Native
- react-native-canvas または Skia
- 共有パッケージ（@repo/vision-core, @repo/shared-types）

### セットアップ（将来）

```bash
cd apps/mobile
npx expo start
```
